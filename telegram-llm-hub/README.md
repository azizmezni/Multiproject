# Telegram LLM Hub

Multi-model AI assistant via Telegram with project boards, automatic fallback, QA testing, and draft management.

## Quick Setup

1. **Create a Telegram Bot** via [@BotFather](https://t.me/BotFather) - send `/newbot` and copy the token
2. Set your bot token in `.env`: `TELEGRAM_BOT_TOKEN=your_token`
3. Install & run:

```bash
cd telegram-llm-hub
npm install
npm start
```

4. Message your bot on Telegram and send `/start`

## Features

- **Multi-LLM Chat** with automatic fallback across providers
- **Reorderable provider list** - drag providers up/down to set priority
- **Project Boards** - Trello-like boards generated by AI from project descriptions
- **Plan Mode** - AI breaks down projects into tasks, asks questions when needed
- **Execute Mode** - runs tasks sequentially with AI assistance
- **QA Testing** - automated CLI tests and vision-based verification per task
- **Draft Board** - share links in Telegram, AI can clone/expand/plan from them
- **Vision Mode** - send photos for AI analysis

## Setting API Keys

**Option 1 - .env file:**
```
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GEMINI_API_KEY=AI...
```

**Option 2 - Telegram commands:**
```
/setkey openai sk-your-key-here
/setkey gemini AIza-your-key-here
```

**Option 3 - Settings UI:**
Use `/settings` > API Keys in Telegram

## How Fallback Works

Providers are tried in priority order (configurable via `/providers`):

```
Claude -> OpenAI -> Gemini -> Mistral -> Groq -> Cohere -> DeepSeek -> Grok -> Ollama -> LM Studio
```

If Claude hits a rate limit, it automatically tries OpenAI, then Gemini, etc. When all cloud providers fail, it falls back to local solutions (Ollama/LM Studio).

## Project Board Flow

1. `/new My E-commerce App` - AI creates a board with tasks
2. AI asks questions about tasks that need clarification
3. Answer all questions -> Execute button appears
4. Execute mode runs tasks one by one
5. After each task, QA tests run automatically
6. Board shows pass/fail status for each task

## Draft Board

Share any link in the chat -> it goes to your draft board with options:
- **Clone as Board** - create a full project board from the idea
- **Plan from this** - generate a detailed plan
- **Expand Idea** - AI brainstorms features and architecture
- **Run CLI Command** - execute related commands
